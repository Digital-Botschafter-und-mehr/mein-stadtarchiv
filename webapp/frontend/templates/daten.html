Daten

Die hier angebotenen Daten unterliegen der freien Creative Commons BY 4.0 Lizenz. Sie können die Dokumente also gerne für eigene Arbeiten wie z.B. Blogbeiträge verwenden, solange Sie einen korrekten Quellenverweis nutzen. Mit dem Kopieren des Quellenverweises unter dem Bild inkl. der Verlinkung sind Sie in jedem Fall auf der sicheren Seite.

Mithelfen

Wenn Sie Ihre Archivdaten ebenfalls auf "Mein Stadtarchiv" veröffentlichen wollen, dann schreiben Sie uns gerne an!

Wenn Sie dagegen einen technischen Fehler auf "Mein Stadtarchiv" finden, tragen Sie ihn bitte in unseren Bugtracker auf GitHub ein. Alternativ senden Sie uns einfach eine E-Mail. Herzlichen Dank!

Open Source

"Mein Stadtarchiv" ist OpenSource-Software unter der 3-Clause BSD License und steht auf GitHub zum Download bereit.
